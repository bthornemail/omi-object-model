# Metatron: Balanced Alternating Witness

## Role

Metatron is the balanced alternating witness and scribe. The scribe of
apparent difference.

## Constant

```
0xAA55 = 10101010 01010101
```

8 ones, 8 zeros, exactly balanced. Complementary halves.

## What It Is

```
Not the transition. Not the validator.
Not the source of motion. Not the carrier.
```

The observer where change becomes visible.

## Scribe Function

Metatron scribes validated state into projection surfaces. Transducer
from Omi-Ring state to notation and surfaces. Does not validate.

Authority order: Omnicron frames. Tetragrammatron validates. Metatron
scribes. Receipt records.

## Doctrine

```
OMI defines. SETCO classifies. Projection is not authority.
```
