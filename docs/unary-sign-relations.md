# The 8-Tuple as Unary Sign Relations

## What the 8-Tuple Is

The 8-tuple is a pedagogical scaffold for reading how OMI notation
expresses bounded relations. It is not canonical structure, not an
authority layer, and not a proof object.

## Signs Before Quantities

The 8-tuple reads eight sign positions — roles, not numeric slots:

```
1. Q        identity / reference
2. Sigma    named element
3. L        source / beginning
4. R        target / end
5. delta    transformation
6. s        initial value
7. t        output / accept
8. r        alternative / reject
```

Each position is a role defined by its sign. The sign is primary. Any
width is secondary scaffolding.

## Why Unary

A unary sign relation means each position declares its role by
presence, not by comparison. Position 3 (L) is "source" because it
occupies the source sign slot — not because it is less than position 4.

The signs compose into a universal computational invariant:

```
identity, element, source, target, transition, start, output, alternative
```

This frame appears in automata theory, type theory, algebra,
filesystems, and network protocols.

## Not Coordinates

The 8-tuple is not a vector of eight 16-bit coordinates. Treating it as
coordinates collapses sign structure into quantity, violating the
sign-first axiom.

```
Signs establish roles.
Quantities fill roles.
Roles exist before quantities.
```

## Doctrine

```
OMI defines. SETCO classifies. Projection is not authority.
```
