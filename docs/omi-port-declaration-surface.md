# OMI-Port: External / Adaptive Declaration Surface

## Role

OMI-Port is the external and adaptive declaration surface for connecting
to external computational environments. OMI-Lisp declares locally.
OMI-Port connects outwardly.

## Ports

```
omi-canon-port            canonical OMI doctrine
omi-lisp-port             local declaration surface
omi-port-port             external/adaptive port surface
omi-isa-port              executable witness rail
omi-protocol-port         legacy protocol language
omi-tetragrammatron-port  bounded validation/governor
omnicron-port             resolver-facing binary core
```

## Authority

```
Port declarations are candidates, not authorities.
Classification precedes promotion.
SETCO classifies ports. OMI defines the port model.
```
