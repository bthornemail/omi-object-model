# Omnicron: Resolver and Projection Surface

## Role

Omnicron is the resolver and projection surface. The axiomatic resolver
for lived device, software, hardware, and world resolution.

```
omni = all-of-OMI / whole-system scope
cron = cadence / chronology / algorithmic clock
```

## Resolver Envelope

```
FF 00 1C 1D 1E 1F 20 FF
```

| Byte | Meaning |
|------|---------|
| FF   | Runtime dialect / resolver horizon |
| 00   | Null origin |
| 1C   | FS (file/frame boundary) |
| 1D   | GS (group/graph boundary) |
| 1E   | RS (record/relation boundary) |
| 1F   | US (unit/symbol boundary) |
| 20   | Readable boundary / earned space |
| FF   | Closure of the same horizon |

## Projection

Every projection is a view, not an authority. Projected views must
remain consistent with originating state.

```
Omnicron frames universally compatible signs.
Projection does not alter validity.
```

## Lived Resolution

Omnicron resolves against real environments — devices, networks,
hardware, runtime contexts. Where deterministic closure meets the world.
