# Tetragrammatron

## Role

Tetragrammatron is the lawful fold governor. Validates exact nibble and
XOR closure conditions.

## Etymology

```
tetra   = fourfold
gramma  = word, mark, grammar, written form
tron    = machine, instrument, computron

Tetragrammatron = tetrahedral grammar computron
```

A grammar machine that validates whether a declared relation closes
into lawful form.

## Fold patterns

```
0x3C = 00111100  (static centered fold)
0x78 = 01111000  (shifted/carried bridge fold)
0x78 = 0x3C << 1
0x3C = 0x78 >> 1
```

## Rules

```
Tetragrammatron carries the fold.
Tetragrammatron validates exact closure.
Tetragrammatron does not frame or resolve.
That belongs to Omnicron.
```
