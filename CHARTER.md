# Charter

## The Omi Object Model

The Omi Object Model is the pure computational paradigm. It owns the
object, address, and relation model for the OMI ecosystem.

## What it defines

```
omi-notation    the canonical notation surface
omi-lisp        the local declaration surface
omi-port        the external/adaptive declaration surface
null ring       the folded null seed and byte-ring closure
gauge ring      the deterministic normalization ruler
Omicron         the deterministic closure surface
Metatron        the balanced alternating witness
Tetragrammatron the lawful fold governor
Omnicron        the resolver and projection surface
```

## What it does not do

```
It does not classify external environments.
It does not stage proposals.
It does not adjudicate promotion status.
It does not teach relationships.
Those belong to SETCO.
```

## Authority order

```
Omnicron frames.
Tetragrammatron validates.
Metatron scribes.
Receipt records.
```

## One-sentence lock

The Omi Object Model is the inside-out pure object/address/relation model
that defines notation, declaration, address, and relation for the OMI
ecosystem without becoming a classification framework.
