# Omnicron

## Role

Omnicron is the resolver and projection surface. It is the axiomatic
resolver surface for lived device/software/hardware/world resolution.

## Resolver envelope

```
FF 00 1C 1D 1E 1F 20 FF
```

| Byte | Meaning |
|------|---------|
| FF   | Runtime dialect / resolver horizon |
| 00   | Null origin |
| 1C   | FS (file/frame boundary) |
| 1D   | GS (group/graph boundary) |
| 1E   | RS (record/relation boundary) |
| 1F   | US (unit/symbol boundary) |
| 20   | Readable boundary / earned space |
| FF   | Closure of the same horizon |

## Etymology

```
omni = all-of-OMI / whole-system scope
cron = cadence / chronology / algorithmic clock
```

## Rules

```
Omnicron resolves. Omnicron does not define.
Omnicron frames universally compatible signs.
Projection is not authority.
```
